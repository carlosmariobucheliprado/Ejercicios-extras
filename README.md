Descripcion de los ejerciciso extras: 
-----------------------------------------------------------------------------------------------
Este taller tiene como objetivo aplicar el manejo de cadenas (String) en Java utilizando diferentes métodos y conceptos fundamentales del lenguaje.

Se desarrollan ejercicios donde se trabajan operaciones como:

  *Creación de cadenas de texto de diferentes formas

  *Comparación de objetos usando == y equals()

  *Manipulación de Strings (charAt, length, substring, replace, etc.)

  *Conversión de mayúsculas y minúsculas

  *Concatenación de cadenas

  *Búsqueda de caracteres dentro de una cadena

  *Inversión de palabras y frases

  *Eliminación de espacios en blanco

  *Conteo de vocales dentro de un texto

Además, se aplican conceptos importantes como:

  *Inmutabilidad de los Strings

  *Diferencia entre comparación por referencia y por contenido
  
------------------------------------------------------------------------------------------------
Instrucciones para ejecutar el programa:
------------------------------------------------------------------------------------------------

1. Clonar el repositorio o descargar el proyecto.

2. Abrir el proyecto en NetBeans.

3. Ubicar el paquete String

4. Ejecutar cualquiera de los siguientes archivos según el ejercicio:

      ViewString.java → Primer ejercicio de Strings

      ViewString2.java → Segundo ejercicio de Strings

5. Para ejecutar:

      Clic derecho sobre el archivo

      Seleccionar Run File

6. El programa se ejecutará en consola y solicitará algunos datos por teclado cuando sea necesario.
